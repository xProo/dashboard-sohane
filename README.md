## Comment exécuter mon application localement ?

Je m'assure d'avoir Node.js installé sur mon ordinateur. Je peux le télécharger et l'installer à partir du site officiel de Node.js.

J'ouvre mon terminal ou ma ligne de commande et je navigue vers le répertoire racine de mon application React.

J'exécute la commande suivante pour installer les dépendances de mon application : npm install

Cette commande me permet de lire le fichier package.json de mon application et d'installer toutes les dépendances nécessaires répertoriées dedans.

Une fois l'installation terminée, je peux lancer mon application en exécutant la commande suivante : npm start

Cette commande démarre le serveur de développement et mon application devient accessible à l'adresse http://localhost:3000 (par défaut).
